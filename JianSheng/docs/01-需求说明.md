# 01 - 需求说明

> 项目：简生 JianSheng 健身打卡微信小程序
> 版本：v1.0 MVP
> 更新日期：2026-07-04

---

## 项目定位

面向个人使用的健身打卡微信小程序，帮助用户记录运动、饮食、饮水，通过日历和目标体系保持健身习惯。

## 目标用户

- 有健身习惯、想要记录和追踪的个人用户
- 对应用复杂度容忍度低，需要简洁直观的操作体验

## V1.0 核心功能（MVP）

### 一、运动打卡记录

- **自定义运动类型**：预设力量、有氧、瑜伽、跑步、游泳、跳绳、球类、居家自重 8 种，支持新增自定义
- **多维度记录项**：组数、次数、重量、时长、距离、配速、心率、动作感受、训练部位（全部可选填）
- **快速模板打卡**：预设常用训练模板，一键完成打卡，10 秒内搞定
- **照片打卡**：上传训练实拍照片，自动添加日期+运动类型水印

### 二、饮食打卡记录

- **三餐+零食记录**：按早餐/午餐/晚餐/零食四个餐别分区记录
- **食物记录**：食物名称 + 热量(kcal) + 照片（可选）+ 备注
- **饮水记录**：点击杯子图标快速记录，可视化当日饮水进度
- **热量合计**：自动合计当日摄入，配合进度条显示目标达成率

### 三、打卡日历 & 目标体系

- **可视化日历**（月视图）：
  - 🟢 绿色 = 当天有打卡记录
  - ⚪ 灰色 = 无记录
  - 🔴 红色 = 连续打卡中断
- **统计数据**：连续打卡天数、本月训练天数、饮食记录天数
- **自定义目标**：每周训练次数、每日饮水量等
- **补卡功能**：错过当天可手动补卡，支持备注断卡原因

## V1.0 不包含（后续版本计划）

- 身体数据记录（体重、体脂、围度）→ V2.0
- 身材对比相册 → V2.0
- 数据趋势图表（体重曲线、围度变化）→ V2.0
- 训练看板、薄弱点 AI 分析 → V3.0
- 热量缺口计算 → V3.0
- 月度健身总结报告/分享 → V3.0
- 拍照识别食物 → V3.0

## 非功能性需求

- 界面简洁直观，操作步骤 ≤ 3 步完成打卡
- 适配 iPhone 和 Android 主流机型
- 数据存储在微信云开发，用户数据隔离
