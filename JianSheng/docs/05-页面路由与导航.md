## 页面清单

### Tab 页面（4 个）

| 页面 | 路径 | 说明 |
|------|------|------|
| 运动打卡 | `pages/workout/workout` | 运动类型选择、详细打卡、模板快速打卡、照片上传 |
| 饮食打卡 | `pages/diet/diet` | 四餐分区、食物记录、饮水记录、热量合计 |
| 打卡日历 | `pages/calendar/calendar` | 月视图日历、打卡统计、目标管理、补卡 |
| 我的 | `pages/mine/mine` | 个人信息、模板管理、身体数据、设置 |

### 子页面（非 Tab，通过 navigateTo 跳转）

| 页面 | 路径 | 来源 |
|------|------|------|
| 模板管理 | `pages/template-manage/template-manage` | 我的 → 训练模板管理 |
| 新建/编辑模板 | `pages/template-edit/template-edit` | 模板管理 → 新建/编辑 |
| 身体数据录入 | `pages/body-data/body-data` | 我的 → 身体数据记录 |
| 目标管理 | `pages/goals/goals` | 日历 → 目标管理 / 我的 → 目标管理 |
| 设置 | `pages/settings/settings` | 我的 → 设置 |

---

## 导航方式

| 场景 | 方法 | 说明 |
|------|------|------|
| Tab 切换 | `wx.switchTab` | 四个主页之间切换 |
| 进入子页面 | `wx.navigateTo` | 从主页进入详情页，可返回 |
| 返回上一页 | `wx.navigateBack` | 左上角返回箭头（微信自动） |

---

## 页面跳转关系

```
运动打卡页 ──→ 无子页面（所有操作在当前页完成）
饮食打卡页 ──→ 无子页面（所有操作在当前页完成）
打卡日历页 ──→ 目标管理页
  我的页   ──→ 模板管理页 ──→ 新建/编辑模板页
           ──→ 身体数据页
           ──→ 目标管理页
           ──→ 设置页
```
