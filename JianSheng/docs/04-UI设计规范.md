# 04 - UI 设计规范

> 项目：简生 JianSheng 健身打卡微信小程序
> 版本：v1.0 MVP
> 更新日期：2026-07-04

---

## 设计理念

**简洁、直观、不费脑。** 打开就能打卡，不用思考。

参考风格：iOS 健康 App、Keep 的清爽感。

---

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | `#4A90D9` | 淡蓝色，按钮、选中态、进度条 |
| 主色调浅 | `#EDF4FC` | 淡蓝背景，卡片底色 |
| 主色调深 | `#3A7BC8` | 按下态、强调文字 |
| 成功/完成 | `#52C41A` | 绿色，日历已打卡标记 |
| 警告/断卡 | `#FF4D4F` | 红色，断卡标记 |
| 文字主色 | `#333333` | 标题、正文 |
| 文字辅色 | `#999999` | 辅助说明、占位文字 |
| 背景色 | `#F5F6FA` | 页面全局背景 |
| 卡片背景 | `#FFFFFF` | 白色卡片 |
| 分割线 | `#EEEEEE` | 列表分割 |

---

## 字体

使用系统默认字体，不引入自定义字体。

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 大标题 | 20px | bold | 页面标题 |
| 标题 | 18px | bold | 卡片标题 |
| 正文 | 15px | normal | 主要内容、列表项 |
| 辅助 | 13px | normal | 说明文字、标签 |
| 小字 | 11px | normal | 日期标记、角标 |

---

## 间距规范

| 元素 | 值 |
|------|-----|
| 页面边距 | 15px |
| 卡片间距 | 12px |
| 卡片内边距 | 15px |
| 列表项间距 | 12px |
| 图标与文字间距 | 8px |
| 表单标签与输入框间距 | 8px |

---

## 圆角 & 阴影

| 元素 | 圆角 | 阴影 |
|------|------|------|
| 页面卡片 | 12px | `0 2px 8px rgba(0,0,0,0.06)` |
| 按钮 | 8px | 无（大按钮加轻微阴影） |
| 输入框 | 8px | 无 |
| 标签/徽章 | 4px | 无 |

---

## 按钮规范

| 类型 | 样式 |
|------|------|
| 主按钮 | 主色背景 + 白色文字，高度 44px，宽度撑满 |
| 次要按钮 | 主色边框 + 主色文字 + 透明背景 |
| 文字按钮 | 主色文字，无背景无边框 |
| 危险按钮 | 红色背景 + 白色文字（删除确认时使用） |

---

## 图标

- 优先使用微信小程序自带图标
- Tab 栏图标：使用 24x24 的 PNG（选中态主色、未选中态灰色）
- 功能图标：使用 emoji 或简单 Unicode 符号

---

## 页面通用布局

```
┌─────────────────────────┐
│     Status Bar          │  ← 微信自动处理
├─────────────────────────┤
│     页面标题（可选）      │
├─────────────────────────┤
│                         │
│   滚动内容区域           │
│   - 白色卡片             │
│   - 15px 边距           │
│                         │
│                         │
├─────────────────────────┤
│  [ Tab Bar ]             │  ← 微信自动处理
└─────────────────────────┘
```

背景色 `#F5F6FA`，内容区域白色卡片式布局。
