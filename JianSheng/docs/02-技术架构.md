## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | 微信小程序原生框架 | WXML + WXSS + JavaScript |
| 后端服务 | 微信云开发 | 免服务器、免域名、免备案 |
| 数据库 | 云数据库（MongoDB 风格） | 文档型数据库，JSON 格式 |
| 文件存储 | 云存储 | 照片/视频上传，自带 CDN |
| 服务端逻辑 | 云函数 | Node.js，处理定时任务等 |
| 开发工具 | 微信开发者工具 | Windows/Mac 桌面应用 |

## 为什么选择这个技术栈？

1. **零运维成本**：不需要买服务器、不需要装数据库、不需要配置域名和 HTTPS
2. **免费额度充足**：云开发个人版免费额度对个人使用绰绰有余
3. **开发效率高**：一个工具搞定前端 + 后端 + 数据库 + 存储
4. **适合新手**：全在微信开发者工具里操作，不需要额外安装 MySQL、Nginx 等

## 数据流

```
用户操作 → WXML 页面 → JS 调用 wx.cloud API → 云数据库/云存储 → 返回结果 → 更新页面
```

## 安全设计

- 云数据库权限：仅创建者可读写（每条数据自动关联用户 openid）
- 小程序端直接操作数据库（V1.0 不需要云函数做后端中转）
- 照片上传限制：单张 ≤ 10MB，仅允许图片格式
