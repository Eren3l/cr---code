# 简生 JianSheng — 健身打卡微信小程序

一款简洁直观的健身打卡微信小程序，帮助你记录每一次运动、每一餐饮食，保持健康的健身习惯。

## 功能

- 🏃 **运动打卡** — 自定义运动类型，多维度记录，快速模板打卡，照片打卡
- 🍽️ **饮食打卡** — 三餐+零食记录，饮水追踪，热量统计
- 📅 **打卡日历** — 可视化日历，连续打卡统计，目标设定，断卡补卡
- 👤 **个人中心** — 训练模板管理，身体数据记录

## 技术栈

微信小程序原生框架 + 微信云开发
